= Redmine Wiki Solution Plugin

== Plugin installation

1. Copy the plugin directory into the vendor/plugins directory
2. Restart Redmine

== Usage

The following macros are added:

* {{show_solution(number_of_solution_showned)}}
* {{start_solution}}
* {{end_solution}}

You can use this plugin to show or hide solutions for an exercice.
